# 🛡️ TryHackMe: Introduction to Cybersecurity Walkthrough

## 🎯 Objective
This module provides a foundational understanding of cybersecurity principles, common attack vectors, and the roles within defensive (Blue Team) and offensive (Red Team) security operations.

---

## 🧭 Core Concepts Covered

### 1. Offensive Security (Red Teaming)
- **What I Learned:** Explored how ethical hackers think. Learned about vulnerability assessment and the basics of unauthorized access simulation.
- **Key Task:** Conducted a basic web exploitation exercise to understand how hackers compromise systems.

### 2. Defensive Security (Blue Teaming)
- **What I Learned:** Focused on protecting assets, detecting threats, and responding to incidents.
- **Key Concepts:** - **Digital Forensics:** Analyzing logs and artifacts to trace a hacker's footprint.
  - **SIEM & Monitoring:** Understanding how security alerts are generated.

### 3. Cyber Careers & Roles
- Explored different career pathways like Security Analyst (SOC), Penetration Tester, and Incident Responder.
